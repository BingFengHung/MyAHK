# MyAHK

AHK in VIM way


### Alt + 1 (移動滑鼠)

- hjkl -> 方向鍵
- ctrl + [hjkl] -> 加速移動
- ctrl + u -> 頁面向上滑動
- ctrl + d -> 畫面向下滑動
- 1 -> 左鍵 click
- 2 -> 右鍵 click
- 3 -> 長按左鍵
- 4 -> 釋放長按左鍵

### Alt + 2 (調整視窗大小與位置)

- hjkl -> 移動式窗的位置
- w -> 視窗寬度加寬
- ctrl + w -> 視窗寬度縮小
- e -> 視窗高度加高
- ctrl + e -> 視窗高度降低
- shift + m -> 視窗置中
- shift + l -> 視窗移動到畫面最右邊
- shift + h -> 視窗移動到畫面最左邊
